# 📊 E-Commerce Sales & Customer Insights Dashboard

An interactive **Power BI dashboard** analyzing e-commerce sales performance and customer demographics.
This project focuses on identifying **sales trends, customer segments, and product performance** using a realistic e-commerce dataset.

---

# 🎯 Project Objectives

The goal of this project is to:

* Analyze **total revenue trends**
* Identify **top performing products**
* Understand **customer demographics**
* Evaluate **regional sales performance**
* Explore **category level sales distribution**

---

# 📊 Key KPIs

The dashboard highlights the following metrics:

| KPI                           | Description                  |
| ----------------------------- | ---------------------------- |
| **Total Revenue**             | Overall sales generated      |
| **Total Orders**              | Total number of transactions |
| **Average Age**               | Average age of customers     |
| **Average Order Value (AOV)** | Average revenue per order    |


# 📈 Dashboard Insights

### Revenue by Age Group

* The **55+ age group** generates the highest revenue.
* Younger age groups contribute less to total sales.

### Revenue by Month

* Sales fluctuate throughout the year.
* Peak months show significantly higher revenue.

### Revenue by Region

* Revenue distribution is relatively balanced across regions.
* **North region** shows slightly higher sales.

### Revenue by Category

* **Electronics dominate total revenue**, representing the majority of sales.

### Top Products by Revenue

Top performing products include:

* Laptop
* Smartphone
* Monitor

### Revenue by Gender

* Male customers contribute a slightly larger portion of total revenue.
